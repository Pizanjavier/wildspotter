## Notas de produccion V3

### Restricciones activas

1. **No ir a spots reales** — todo el contenido usa stock footage generico del tipo de paisaje
2. **No revelar ubicaciones** — ni coordenadas, ni municipio, ni provincia especifica. Solo tipo de paisaje + datos
3. **Datos borrosos** — capturas del pipeline/mapa siempre pixeladas o parciales

### Render de carousels

```bash
# Legal (6 slides)
for i in $(seq 1 6); do npx remotion still Carousel-Legal-$i --image-format=png out/carousel-legal-$i.png; done

# Pipeline (9 slides)
for i in $(seq 1 9); do npx remotion still Carousel-Pipeline-$i --image-format=png out/carousel-pipeline-$i.png; done

# Scoring (6 slides)
for i in $(seq 1 6); do npx remotion still Carousel-Scoring-$i --image-format=png out/carousel-scoring-$i.png; done

# Mystery (5 slides)
for i in $(seq 1 5); do npx remotion still Carousel-Mystery-$i --image-format=png out/carousel-mystery-$i.png; done
```

Los conceptos M (mystery reveal) y O (quiz legal) necesitan componentes Remotion. Se produciran con el `remotion-producer` agent cuando se prioricen.
